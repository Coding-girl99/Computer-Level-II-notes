# PPT 基本操作

- 1px = 25cm。

- 搜索“母版”或者，在设计/视图选项卡 -> 点击编辑母版，点击编辑模板，可以设置统一样式。

- 插入幻灯片编号：插入选项卡 -> 幻灯片编号/页眉页脚 -> 勾选幻灯片编号

	注意，如果勾选标题幻灯片不显示，那第一张幻灯片就会从2开始，怎么解决呢？

	设计选项卡 -> 幻灯片大小 -> 自定义大小 -> 修改幻灯片编号起始值为 0。

- 在PPT幻灯片里链接外部文档：插入选项卡 -> 对象 -> 插入文档，勾选链接。

- 既要设置字体样式又要设置字号间距的时候，先设置样式再设置字号。

- 设置文本框位于幻灯片正中间：选中文本框 -> 绘图工具选项卡 -> 对齐 -> 设置水平居中、垂直居中。

- 动画设置：开始：从上一项开始=之前，**即和上一个动画同时开始**；在上一项之后开始=之后，即在上一个动画之后开始。

- 分清主标题、副标题、文本框，选对要添加动画的对象。

- 幻灯片设置开始：之前，延迟1s和开始：之后，延迟1s的区别：计时起始点不同。

	- 之前就从上一个动画出来开始计时。
	- 之后就是等上一个动画完全展示完过后再计时。
